# Ledger-Wealth-Managment
💰 Offline-first personal finance PWA. Scan receipts, UPI screenshots &amp; bank statements (PDF/CSV/XLSX) — AI reads, categorizes &amp; sorts every transaction automatically. Live spending trajectory, AI-written insights &amp; saving tips. 100% on-device data, no server, no account, no ads. Built with React, TypeScript &amp; Dexie.js.
